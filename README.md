# PopKwiz

## Development

[Firebase authentication keys must be present](https://firebase.google.com/docs/admin/setup#initialize-sdk) within file `keys.json` in the root folder.

```shell
$ npm install

$ npm run dev-client

$ npm run dev-server
```

## Production
```shell
$ npm install

$ npm run build

$ npm start
```
